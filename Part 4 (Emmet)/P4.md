# Emment

## Starter Shortcut
```
! + ENTER
```
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
### Generating HTML Tags
```
tag_name + ENTER
```
```
<tag_name)></tag_name>
```
This also fills certains tags w/ common attributes.
